# 🏧 ATM Management System (Python)

A simple **ATM Management System** developed using **Python**. This project simulates basic ATM operations through a console-based interface and is ideal for beginners learning Python programming concepts.

## 🚀 Features

* User Login with PIN
* Balance Inquiry
* Cash Deposit
* Cash Withdrawal
* PIN Verification
* Exit Option
* Menu-driven interface

## 🛠️ Technologies Used

* Python 3
* VS Code

## 📂 Project Structure

```text
ATM/
├── atm.py
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ATM-Management-System.git
   ```

2. Navigate to the project folder:

   ```bash
   cd ATM-Management-System
   ```

3. Run the program:

   ```bash
   python atm.py
   ```
